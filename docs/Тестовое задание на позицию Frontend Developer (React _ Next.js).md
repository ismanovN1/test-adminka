# **Тестовое задание на позицию Frontend Developer (React / Next.js)**

## **Общая информация**

Необходимо разработать современную административную панель (Analytics Dashboard), максимально приближенную к реальному коммерческому SaaS-продукту.

Основная цель задания — оценить не только знание React, но и уровень проектирования архитектуры, качество кода, умение работать с большим количеством данных, производительность приложения и внимание к деталям.

**Рекомендуемое время выполнения:** 10–15 часов.

# **Технологический стек**

### **Обязательно**

* Next.js 15 (App Router)  
* React  
* TypeScript  
* Tailwind CSS  
* TanStack Query  
* Axios  
* Zustand  
* React Hook Form  
* Zod

### **Будет преимуществом**

* Framer Motion  
* Recharts или Apache ECharts  
* React Virtuoso / TanStack Virtual  
* next-intl  
* PWA
